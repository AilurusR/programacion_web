HTML

¿Qué es HTML?

  • Es un lenguaje de marcado de hipertexto (Hyper Text Markup Lenguaje).
  • HTML no es un lenguaje de programción, es un lenguaje de estructura.
  • Cada etiqueta le dice al navegador y a los motores de búsqueda cuál es la estructura de los documentos, elementos, organización, etc.

Breve historia de HTML

  • 1989: Inicio de su desarrollo.
  • 1991: Lanzamiento de la web (se basa en tres conceptos: http, html y url).
  • 1992: Lanzamiento de HTML (Como precursor tuvo a SMGL, creado por Tim Bernes Lee).
  • 1994: Creación de la W3C (Consorcio que define los estandares de la web).
  • 1998: HTML 4.
  • 1999: HTML 4.1 / XHTML. (Actualización de html 4 pero con una notación más rigida basada en xml).
  • 2004: Creación de la WHATWG, consorcio que tomo el proyecto html 4 (de forma independiente) y lo continuo.
  • 2008: HTML 5, lanzado por WHATGW.
  • 2014: Estandar de HTML 5, lanzado de forma oficial por W3C.

Estructura basica de una etiqueta:

Todo en HTML es una etiqueta, y cada etiqueta sirve para colocar contenido especifico. Asi la etiqueta <img> es la etiqueta destinada a contener un recurso de tipo imagen. En su mayoria las etiquetas estan compuestas de tres partes:

  • Etiqueta de apertura.
  • Contenido.
  • Etiqueta de cierre.
<h1>
Contenido de la etiqueta.
</h1>

Estructura basica de un documento HTML:

Cada pestaña o ruta de una página web es un documento de texto plano con extención ".html", por convención a la ruta raíz (es decir, la primer página que se carga al acceder a dicha pagina web) se suele llamar "index.html". Esta convención se respeta ya que los navegadores web por default buscan y cargan este archivo al entrar en dicha página web. La estructura de un archivo .html sin contenido, o recien creado se ve de la siguien forma:

<!DOCTYPE html>
<html lang="en">
<head>
<title>Document</title>
</head>
<body>
</body>
</html>

Los significados de estas etiquetas son los siguientes:

  • !DOCTYPE html: Esta etiqueta indica el tipo de documento. Es un preámbulo requerido. Los tipos de documento actúan como vínculos a un conjunto de reglas que el código HTML de la página debía seguir para ser considerado bueno, lo que puede significar la verificación automática de errores y algunas otras cosas de utilidad.
  • html: Este elemento encierra todo el contenido de la página entera y, a veces, se le conoce como el elemento raíz (root element). El atributo "lang="en"" indica el idioma en que estará el contenido de la página.
  • head: Este elemento actúa como un contenedor de todo aquello que quieres incluir en la página HTML que no es contenido visible por los visitantes de la página. Incluye cosas como palabras clave (keywords), una descripción de la página que quieres que aparezca en resultados de búsquedas, código CSS para dar estilo al contenido, declaraciones del juego de caracteres, etc.
  • title: Establece el título de tu página, que es el título que aparece en la pestaña o en la barra de título del navegador cuando la página es cargada
  • body: Encierra todo el contenido que deseas mostrar a los usuarios web que visiten tu página, ya sea texto, imágenes, videos, juegos, pistas de audio reproducibles, y demás.

Comentarios en HTML

Los comentarios en html se indican de la siguiente manera:
<!--"el comentario que quieras poner..."-->

La utilidad de los comentarios radica en dejar información util para ti u otras personas que ayude a entender el código que acabas de escribrir, quién lo escribio u otras cosas.
